UPDATE LOG

20.03.2024 v.0.0.2.1 (Bug fixing build)

Hello. Yesterday's build was veeeeery buggy, so I decided to make this Bug Fixing build. Besides fixing bugs I also added and improved some things. You can read about it below

Fixed:

- Buttons on some mobile devices were scaling really bad. I fixed it

- When the player picked up an object (in our case, a hammer) and then crouched while holding the object, the Y size of the object decreased twice. I fixed this bug

- When the player picked up the object (hammer) and then looked down, he flew away on the hammer (seriously XDD)(picture at the bottom of the page). Of course I fixed this bug

- Pick up button was visible while player was looking at any object. I fixed this bug

- When the player used the drop button, the item was able to fell off the map. I fixed this bug

Added:

- Second hammer to test pickup/drop system

- Light Bulb

Improvements:

- Changed pickup/drop system. Now when player picks up an item, it appears at player's right hand. While crouching, item doesn't changes it's size. Now while player drops item, it gets dropped in front of the player (in version 0.0.2 it was simply dropped under the player). Also when player wants to pick up another item (while already holding other item), player drops item which he has in his hand and picks up item.

- Now we have 1 room in the game! It doesn't has any furniture etc. inside (besides light bulb), but I'll decorate it in the future builds!

- Changed door opening/closing animation (just a bit)
